## Мета проєкту: Поглибити знання про класи в C#, навчитися проєктувати систему з кількох взаємопов'язаних класів. Практикувати принципи композиції об'єктів та розподілу відповідальності між класами на прикладі симуляції дуельного клубу.

## Опис  
- Проєкт реалізований на C#. У репозиторії є файл рішення (`.sln`), файл проєкту (`.csproj`) та вихідні `.cs`-файли.
- Основні компоненти:  
  - `Entities.cs` — опис сутностей / моделей даних.
  - `HogwartsDbContext.cs` — контекст бази даних (якщо застосовується база або ORM).
  - `Program.cs` — точка входу у програму. 
- Проєкт може слугувати як приклад структури, роботи з моделями/даними, або як база для подальших розширень.

## Як запустити  
- Склонувати репозиторій.
- Відкрити рішення DuelingSimulation.sln або проект DuelingSimulation.csproj у IDE, що підтримує C#.
- Зібрати (Build) проект.
- Запустити програму — виконуватиметься код з Program.cs.

## Структура каталогу
  DuelingSimulation.sln — рішення
  DuelingSimulation.csproj — проєкт
  Program.cs — точка запуску
  Entities.cs — класи/моделі сутностей
  HogwartsDbContext.cs — контекст бази даних
